# DigitalReminders – Sito istituzionale

Questo repository contiene il sito statico ufficiale di **DigitalReminders**.

## Contenuti
- **Homepage**: presentazione del servizio di reminder automatici via SMS e WhatsApp.
- **Privacy Policy**: informazioni sul trattamento dei dati personali degli utenti.
- **Termini di Servizio**: condizioni di utilizzo della piattaforma.

## Struttura
digitalreminders-site/
│
├── index.html        # Homepage
├── privacy.html      # Privacy policy
├── terms.html        # Termini di servizio
├── style.css         # Stile CSS minimale
└── assets/
└── logo.png      # Logo aziendale

## Deploy
Il sito è statico (HTML + CSS) e può essere pubblicato facilmente su:
- [Render](https://render.com/) (Static Site)
- [GitHub Pages](https://pages.github.com/)
- Qualsiasi altro hosting statico

## Note
Questo sito serve principalmente a:
1. Verifica del dominio su **Meta Business Manager**.
2. Dare un riferimento istituzionale ai clienti.
3. Fornire trasparenza su termini e privacy.
